Project Ideas

My project idea is creating a FAQ and implementing it as a chatterbot. It will be presented in the form of an application and will support both text input and voice input. The chatterbot will have the ability to chat with the user, answer the user's questions or execute the user's command. It not only can be simply used as a chatterbot to relieve the users' stress but also can be ported to another platform. For example, it can perform as online customer service on a website, respond to the players in a game and even give some advice to help the users making an important decision.

The reason why I want to do the project is very simple: It is the project I am most likely to accomplish in the near future. Of course, I want to develop a game which is also my dream, but the truth is that I'm just on my way and I still have a long way to go. So currently, I only want to start with a chatterbot which is very easy to implement since it is not an AI, I can even do it using Java. The chatterbot is undoubtedly very useful and can be used in everywhere, every field to change the people's life.

The chatterbot will be able to customize, self-optimize and self-regulate.

First of all, as a chatterbot application, it has the basic function of chat, and according to the users' requirement, It will be customized into different versions. The version used for games can answer the players' questions about the game, tell them how to complete the task, guide the players to become stronger, collect the players' feedback. The version used for navigation can show the users the information they want in a website, tell the users what they can do or should do when they are confused. The version used for customer service can reply to the users according to their needs, give them advice when they need, answer their questions and collect feedback. The version used for family can help the parents take care of their children, answer the questions about the homework. All in all, the chatterbot will be easily customized according to the user's need.

Secondly, It will have the ability to self-optimize. The program will have a corpus and it will use the information collected from the users to update the corpus to perform better. It will also download information online and use up-to-date information to respond to the user. If the user is satisfied with the response, it will save the information into its database, otherwise, it will delete it. The data will also be updated frequently to avoid error and what the engineers need to do is fixing the potential bugs and add some features according to the users' needs.

The most powerful and difficult feature of this program is self-regulation. The program will have the ability to feel the moods of the users and make different reactions to different users. For example, It will never joke with a person who is sad and it will try to soothe the user's emotion when the user is angry. Just like big data, It will provide different information to different people, identify the signs of satisfaction and dissatisfaction, select information according to users' preference. It can be used for counseling and can even help people with depression because it can always find the best words to reply immediately, no matter when or where.

The chat part of the program will be very easy to implement, all we need is just a computer and a speech recognition system but the other feature will be quite difficult to achieve. Currently, I have no ideas about how to achieve self-optimize and self-regulate, it may require very complicated logic algorithm or this will be more like an AI, need the help of a biological expert. It may require some future technology to help implement these features and It may require cloud computing to support its operation.

It requires at least one computer language like Java to complete the framework of the program but it can only handle the hardcoded part which is designed by the programmer and it may require other computer languages like SQL to built the database, Python or C++ to achieve more complex arithmetic logic. It also requires the skills about the recurrent neural network to identify the users' emotions, the skills to create dynamic memory networks, skills about LexRank, TextRank to answer the users' questions. The system optimization skill will be also needed to make this app available to multiple platforms.

If the project is successful, it will change people's lives in many ways. The most significant change is that we will have a reliable robot friend who can answer almost all the questions and listen patiently to our thoughts. Some jobs like Customer services will disappear slowly but there will be a broader employment space for programmers, engineers and salesman. People with mental illness like depression will get better treatment since there will always be someone who can listen to their hearts. The world will be more efficient and harmonious.
